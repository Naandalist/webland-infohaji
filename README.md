# Info Haji

Landing page untuk aplikasi Info Haji - Aplikasi untuk cek nomor porsi haji dan informasi keberangkatan.

## Setup

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

## Tech Stack

Astro, React, Tailwind CSS
